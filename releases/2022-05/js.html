App Configuration 1.4.0-beta.1 Changelog

Features Added

  • Migrated from using @azure/core-http to depend on newer version of Core libraries @azure/core-client and @azure/core-rest-pipeline which bring better maintainability and performance. #20766. See Azure Core v1 vs v2 for more on the difference and benefits of the move.
  • As part of the migration to @azure/core-client and @azure/core-rest-pipeline, through the operation options, the methods such as AppConfgurationClient#addConfigurationSetting can take a callback (onResponse) to access the HTTP response. See Change to the _response property.

Bugs Fixed

  • Fix an issue where React-Native is loading the wrong file. Adding a react-native mapping to point to the ESM entrypoint file. PR #21119

Artifacts 1.0.0-beta.10 Changelog

Features Added

Added LinkConnectionOperations

Avro 1.0.0 Changelog

Other Changes

  • Compatability for old payload format has been removed.
  • Errors may include a cause field that stores inner errors if any.

Azure Core Http Compat 1.3.0 Changelog

Features Added

  • Introduces CompatResponse type for the _response property that is compatible with the response in @azure/core-http. Please refer #20766 for further details.

Azure Video Analyzer Edge 1.0.0-beta.4 Changelog

Other Changes

Deprecated. We’re retiring the Azure Video Analyzer preview service, you’re advised to transition your applications off of Video Analyzer by 01 December 2022. This SDK is not longer maintained.

Blobs 12.10.0-beta.1 Changelog

Features Added

  • Added support for service version 2021-06-08.
  • Added pageable output of PageBlobClient.listPageRanges() and PageBlobClient.listPageRangesDiff().
  • Added ability to copy source blob tags for BlobClient.syncCopyFromURL().

Bugs Fixed

  • Fix incorrect browser mapping path for BufferScheduler.js
  • Add react-native mapping to ESM entrypoint

Blobs 12.10.0 Changelog

Features Added

  • Includes all features released in 12.10.0-beta.1.

Bugs Fixed

  • Refined user-agent value to avoid failure when os information is not available on some platforms.
  • Fix an issue of not returning raw blob properties in ContainerClient.listBlobsFlat() and ContainerClient.listBlobsByHierarchy().

Blobs ChangeFeed 12.0.0-preview.3 Changelog

Features Added

-Added support for event schema V3, V4, and V5.

Core - Client - Abort Controller 1.1.0 Changelog

  • Changed TS compilation target to ES2017 in order to produce smaller bundles and use more native platform features
  • With the dropping of support for Node.js versions that are no longer in LTS, the dependency on @types/node has been updated to version 12. Read our support policy for more details.

Core - Client - Client 1.6.0 Changelog

Features Added

  • Added a new property endpoint in ServiceClientOptions and mark the baseUri as deprecated to encourage people to use endpoint. See issue link here
  • Upgraded our @azure/core-tracing dependency to version 1.0
  • Add callbacks to support Storage challenge authentication PR#21678

Core - Client - Core Rest Pipeline 1.8.1 Changelog

Features Added

  • Support TLS Settings for client certificate authentication. #21172

  • Exposed type guard for RestError called isRestError for typesafe exception handling.

  • Improve user agent information for React-Native.

Core - Client - Core Utils 1.0.0 Changelog

Features Added

  • Add helpers isObject, isError, getErrorMessage for handling unknown Error objects.
  • Add helper getRandomIntegerInclusive for randomly selecting a whole integer value from a given range.

Core - Client - HTTP 2.2.5 Changelog

Bugs Fixed

  • Fix an issue where React-Native is loading the wrong file. Adding a react-native mapping to point to the ESM entrypoint file. PR #21118
  • delay creating XML parser/builder objects so that packages not requiring XML functionality but running on platforms lacking XML support can still load this package. PR #21118
  • Add a react-native mapping to use xml2js as it is already in our dependency list. Customer can get it to work after installing stream and timers packages in their React-Native project.
  • Fix a run-time error in user agent policy when running in React-Native.

Core - Client - Tracing 1.0.1 Changelog

Other Changes

  • Minor improvements to the typing of updatedOptions when calling startSpan.

Core - Client - XML 1.2.1 Changelog

Bugs Fixed

  • Add a react-native mapping to use fast-xml-parser for xml building/parsing as it is already in our dependency list. With this customers don’t need to polyfill the DOM api for XML processing.

Event Grid 4.10.0-beta.1 Changelog

Features Added

  • Add support for specifiying a channel to send to in a specific Partner Topic via the channelName property of SendOptions.

Event Hubs 5.8.0 Changelog

Breaking Changes

Files Shares 12.10.0-beta.1 Changelog

Features Added

  • Added support for service version 2021-06-08.
  • Added ability to maintain a file’s current last write time when calling ShareFileClient.uploadRange() and ShareFileClient.uploadRangeFromURL().
  • Added ability to specify a file or directory’s change time when calling ShareFileClient.create(), ShareDirectoryClient.create(), ShareFileClient.setProperties(), ShareDirectoryClient.setProperties(), ShareFileClient.resize(), ShareFileClient.rename() and ShareDirectoryClient.rename().
  • Added ability to specify content-type on a file when calling ShareFileClient.rename().

Bugs Fixed

  • Add react-native mapping to ESM entry point
