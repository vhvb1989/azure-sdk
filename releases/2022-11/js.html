Release highlights

Administration 4.4.0-beta.1 Changelog

Features Added

  • Added KeyVaultSettingsClient to get and update Managed HSM settings.
  • Added support for service version 7.4-preview.1.

Blobs 12.12.0 Changelog

Features Added

  • Includes all features released in 12.12.0-beta.1.

Bugs Fixed

  • Fixed an issue of escaping slashes in blob name unnecessarily.

Common 1.0.0-beta.1 Changelog

Features Added

  • Initial release of the keyvault-common package. The package is intended for internal consumption only and only exposes helpers and other common code used by the Key Vault SDKs.

Communication Common 2.2.0 Changelog

Bugs Fixed

  • Fixed the logic of getIdentifierRawId and createIdentifierFromRawId for PhoneNumberIdentifier to always maintain the original phone number string whether it included the leading + sign or not.

Features Added

  • Optimized the logic for deserializing types derived from the CommunicationIdentifier.

Core - Client - AMQP 3.2.0 Changelog

Bugs Fixed

  • Fix React-Native bundling issue by adding a react-native mapping to ESM entrypoint file. (PR #23524)[https://github.com/Azure/azure-sdk-for-js/pull/23524]

Core - Client - Core Rest Pipeline 1.10.0 Changelog

Bugs Fixed

  • Fixed an issue where policies added to a cloned Pipeline would also be added to the original (and vice versa) until policies were removed from either. #23316

Core - Client - HTTP 2.3.0 Changelog

Features Added

  • Add support for x-ms-text PR# 23631

Core - Client - HTTP 2.3.1 Changelog

Bugs Fixed

  • Undo PR #23660 so that released listing blobs works again in storage-blob.

Core - Paging 1.4.0 Changelog

Features Added

  • Add toElements function to convert a page to a list of elements

DevCenter 1.0.0-beta.1 Changelog

Features Added

Initial release of the Azure DevCenter package

Event Grid 4.11.0 Changelog

Features Added

  • Added new System Events:

  • Microsoft.HealthcareApis.DicomImageCreated
  • Microsoft.HealthcareApis.DicomImageDeleted

Files Data Lake 12.11.0 Changelog

Features Added

  • Includes all features released in 12.11.0-beta.1.

Bugs Fixed

  • Fixed an issue of escaping slashes in file or directory path unnecessarily.

Files Shares 12.12.0 Changelog

Features Added

  • Added support for service version 2021-10-04.

Bugs Fixed

  • Fixed a hang issue in ShareFileClient.downloadToBuffer when encountering transient network failure.
  • Refined URL parsing method to let it be able to correctly parse URLs with account name in path.
  • Fixed an issue of escaping slashes in file or directory path unnecessarily.

Identity 3.0.1 Changelog

Bugs Fixed

  • Fixed bug to enable additionallyAllowedTenants to pass through to MSAL Flow.

Identity 3.1.0 Changelog

Other Changes

  • Docs improvements for cross-language alignment

Keys 4.7.0-beta.1 Changelog

  • Added support for Octet Key Pair (OKP) key types as well as sign, verify, signData, and verifyData cryptography operations when using OKP keys.
  • Added support for the Ed25519 curve for the creation of OKP keys.

Load Testing 1.0.0-beta.1 Changelog

Features Added

Initial release of the Azure LoadTesting package

Maps Common 1.0.0-beta.2 Changelog

Bugs Fixed

  • Correct the path of type declation files.

Maps Route 1.0.0-beta.1 Changelog

Features Added

  • Initial Release

Maps Search 1.0.0-beta.2 Changelog

Bugs Fixed

  • Correct the type declaration file generation.

Monitor OpenTelemetry Exporter 1.0.0-beta.9 Changelog

Breaking Changes

  • Azure Monitor OpenTelemetry Metrics Exporter Configuration updated.

Bugs Fixed

  • Suppress tracing while exporting metrics.
  • Envelopes not populating sampleRate correctly.

Features Added

  • Application Insights Sampler.
  • Added retriable behavior for 502, 503 and 504 status codes.
  • Export Metric attributes and Histogram Min/Max values.
  • Added new config options disableOfflineStorage, storageDirectory and exposed ApplicationInsightsClientOptionalParams for HTTP client extra configuration.

Monitor OpenTelemetry Exporter 1.0.0-beta.10 Changelog

Features Added

  • Add network Statsbeat Metrics.

OpenTelemetry Instrumentation 1.0.0-beta.2 Changelog

Features Added

  • Added the ability to disable Azure SDK Spans from being recorded by setting the AZURE_TRACING_DISABLED environment variable to true.
  • Added support for AZURE_HTTP_TRACING_CHILDREN_DISABLED environment variable which allows disabling all children of our core HTTP spans from being recorded.

Queues 12.11.0 Changelog

Bugs Fixed

  • Refined URL parsing method to let it be able to correctly parse URLs with account name in path.

Resource Management - Azure VMware Solution 3.2.0 Changelog

Resource Management - Chaos 1.0.0-beta.2 Changelog

Resource Management - Data Factory 10.9.0 Changelog

Resource Management - Devhub 1.0.0-beta.1 Changelog

Resource Management - Elasticsan 1.0.0-beta.1 Changelog

Resource Management - Hybridcontainerservice 1.0.0-beta.1 Changelog

Resource Management - Nginx 2.0.0 Changelog

Resource Management - Policy Insights 6.0.0-beta.2 Changelog

Resource Management - Redis Cache 7.1.0 Changelog

Service Bus 7.7.3 Changelog

Bugs Fixed

  • Fix React-Native bundling issue by adding a react-native mapping to ESM entrypoint file. (PR #23524)[https://github.com/Azure/azure-sdk-for-js/pull/23524]

$> npm install @azure-rest/maps-route@1.0.0-beta.1
